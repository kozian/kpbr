# О руководствах
Здесь будут собраны руководства по обновлению и настройке конкретных роутеров. 

Как самый оптимальный вариант предлагается роутер `CUDY TR3000` (он же `CUDY TR30`). 
Он имеет достаточную производительность, 128mb хранилища что позволяет поставить много разных пакетов, включая sing-box и podkop.
